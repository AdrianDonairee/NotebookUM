🧪 Test Manual - Validador de PDF

📋 Firma de Archivo PDF

Hex: 25 50 44 46

ASCII: %PDF

Offset: 0 (inicio del archivo)

Fuente: Wikipedia - File Signatures

Subir Archivo para Validar



📝 Instrucciones:

  1. Asegúrate de que el servidor Flask esté corriendo: python app.py
  2. Selecciona un archivo (PDF o cualquier otro tipo)
  3. Haz clic en "Validar Archivo"
  4. Verifica el resultado de la validación

✅ Casos de Prueba Sugeridos:

  • Archivo PDF válido (.pdf) - Debería retornar is_valid: true
  • Imagen PNG (.png) - Debería retornar is_valid: false
  • Archivo de texto (.txt) - Debería retornar is_valid: false
  • Archivo vacío - Debería retornar is_valid: false
